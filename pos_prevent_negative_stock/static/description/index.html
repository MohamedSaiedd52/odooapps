POS Prevent Negative Stock Module

POS Prevent Negative Stock Configuration to configure the module settings.

Steps to Configure POS Prevent Negative Stock

  1. Install the POS Prevent Negative Stock module in your Odoo instance.
  2. Navigate to the Point of Sale (POS) interface.
  3. Select a product with a stock quantity of zero.
  4. Attempting to add this product to the POS order will be restricted.
  5. This restriction ensures that no products with insufficient stock (quantity 0 or negative) can be sold.
  6. The module enforces accurate stock management by ensuring that no sales are made unless the stock is sufficient for the transaction.
