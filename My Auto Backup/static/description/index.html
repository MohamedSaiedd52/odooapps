Database Backup Configuration Menu

Got Setting --> Technical --> Backup Configuration to configure backups

Create New Database Backup Configuration

Enter the database name and master password. Specify the backup type and destination. Enter the backup directory path. If the directory does not exist, a new directory will be created.

Enabling Developer Mode and Backup Setup Steps

After installation, you need to enable developer mode to access the "Technical" menu. Inside "Technical," you will find "Database Backup" under "Configurations." Here, you need to:

  • Enter the actual host and database name from which you want to take a backup.
  • Specify a directory path for the backup. Ensure the folder has the correct permissions by running: chown odoo:odoo.

Then, go to "Scheduled Actions" and find the cron job named Backup: Automatic Database Backup. Adjust its settings to define how often backups should be taken.
