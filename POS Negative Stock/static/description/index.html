POS Prevent Negative Stock Module

POS Prevent Negative Stock Configuration to configure the module settings.

Steps to Configure POS Prevent Negative Stock

  1. Install the POS Prevent Negative Stock module in your Odoo instance.
  2. Enable Developer Mode in your Odoo instance.
  3. Navigate to Settings --> Technical --> POS Prevent Negative Stock under Configurations.
  4. Configure the settings to restrict sales for products with insufficient stock:
    • Enable Stock Restrictions: Toggle to prevent sales of products with negative stock.
    • Notification Options: Choose how to notify users when stock levels are insufficient.
  5. Save the configuration.
  6. Test the configuration by attempting to sell a product with insufficient stock.
